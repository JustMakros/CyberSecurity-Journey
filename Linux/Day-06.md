# Linux Day 06

📅 **Дата:** 28.06.2026

## 🛠 Практическая работа №2

### 🎯 Цель

Научиться создавать и редактировать текстовые файлы в терминале Linux с помощью редакторов **Nano** и **Vi**, сохранять изменения и выходить из редакторов различными способами.

### ✅ Выполненные действия

1. Перешёл в домашний каталог пользователя.
2. Создал рабочую директорию `editors_practice`.
3. Создал файл `file_nano.txt` в редакторе Nano.
4. Ввёл текст в файл.
5. Сохранил изменения с помощью `Ctrl + O`.
6. Подтвердил имя файла клавишей `Enter`.
7. Вышел из редактора с помощью `Ctrl + X`.
8. Проверил содержимое файла командой `cat`.
9. Открыл встроенную справку редактора Nano.
10. Создал файл `file_vi.txt` в редакторе Vi.
11. Перешёл в режим вставки.
12. Ввёл текст в файл.
13. Вернулся в командный режим.
14. Сохранил файл командой `:w`.
15. Вышел из редактора командой `:q`.
16. Повторно открыл файл и добавил новую строку.
17. Сохранил изменения и вышел командой `:wq`.
18. Проверил итоговое содержимое файла.
19. Проверил выход без сохранения командой `:q!`.
20. Просмотрел содержимое рабочего каталога.
21. Вернулся в домашний каталог.
22. Удалил рабочую директорию.

### 📂 Полученная структура

```text
editors_practice/
├── file_nano.txt
├── file_vi.txt
└── test_exit.txt
```

> После выхода через `:q!` файл `test_exit.txt` может отсутствовать или быть пустым.

### 📸 Рисунок 2

**Работа в редакторе Vi**

> Вставить скриншот перехода в режим вставки (`i`), сохранения файла (`:w`, `:wq`) и выхода без сохранения (`:q!`).

### 💻 Изученные команды

| Команда    | Назначение                               |
| ---------- | ---------------------------------------- |
| `nano`     | Открытие текстового редактора Nano       |
| `Ctrl + O` | Сохранение файла в Nano                  |
| `Ctrl + X` | Выход из Nano                            |
| `Ctrl + G` | Открытие справки Nano                    |
| `vi`       | Запуск редактора Vi                      |
| `i`        | Переход в режим вставки                  |
| `a`        | Добавление текста после текущего символа |
| `Esc`      | Возврат в командный режим                |
| `:w`       | Сохранение файла                         |
| `:q`       | Выход из редактора                       |
| `:wq`      | Сохранение файла и выход                 |
| `:q!`      | Выход без сохранения                     |
| `cat`      | Просмотр содержимого файла               |
| `ls`       | Просмотр содержимого каталога            |
| `rm -r`    | Удаление каталога вместе с содержимым    |
