## 🛠 Практическая работа №1

### 🎯 Цель

Научиться создавать жёсткие и символические ссылки, определять их свойства, анализировать индексные дескрипторы (inode) и понимать различия между двумя типами ссылок.

### ✅ Выполненные действия

1. Перешёл в домашний каталог пользователя.
2. Создал рабочую директорию `links_practice`.
3. Создал тестовый файл `file1`.
4. Создал жёсткую ссылку `hard_link_file1`.
5. Создал символическую ссылку `sym_link_file1`.
6. Просмотрел inode файлов командой `ls -i`.
7. Получил подробную информацию о файлах с помощью `ls -li`.
8. Проверил количество жёстких ссылок на файл.
9. Создал дополнительную жёсткую ссылку `hard_link_file2`.
10. Повторно проверил количество ссылок.
11. Записал текст в исходный файл.
12. Проверил содержимое файла через жёсткую ссылку.
13. Проверил содержимое файла через символическую ссылку.
14. Удалил исходный файл.
15. Проверил работу жёсткой ссылки после удаления исходного файла.
16. Проверил работу символической ссылки после удаления исходного файла.
17. Просмотрел информацию о битой символической ссылке.
18. Удалил созданные ссылки.
19. Вернулся в домашний каталог.
20. Удалил рабочую директорию.

### 💻 Изученные команды

| Команда  | Назначение                                        |
| -------- | ------------------------------------------------- |
| `ln`     | Создание жёсткой ссылки                           |
| `ln -s`  | Создание символической ссылки                     |
| `ls -i`  | Просмотр inode файлов                             |
| `ls -li` | Подробная информация с inode и количеством ссылок |
| `cat`    | Просмотр содержимого файла                        |
| `echo`   | Запись текста в файл                              |
| `rm`     | Удаление файлов                                   |
| `rmdir`  | Удаление пустой директории                        |
