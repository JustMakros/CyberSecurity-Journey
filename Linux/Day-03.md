### 💻 Изученные команды

| Команда    | Назначение                                                     |
| ---------- | -------------------------------------------------------------- |
| `history`  | Просмотр истории команд                                        |
| `!!`       | Повтор последней команды                                       |
| `!номер`   | Выполнение команды по её номеру                                |
| `!команда` | Выполнение последней команды, начинающейся с указанного текста |
| `help`     | Встроенная справка Bash                                        |
| `man`      | Руководство по использованию команд                            |
| `whatis`   | Краткое описание команды                                       |
| `whereis`  | Поиск расположения программы и документации                    |
| `apropos`  | Поиск команд по ключевому слову                                |
