Изученные команды
whoami — отображение текущего пользователя.
pwd — отображение текущего каталога.
cd — переход между каталогами.
ls — просмотр содержимого каталога.
ls -a — отображение скрытых файлов.
ls -l — подробная информация о файлах.
ls -la — полный подробный список файлов.
