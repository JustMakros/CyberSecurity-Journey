### 🎯 Цель

Научиться определять текущего пользователя, своё расположение в файловой системе и анализировать содержимое домашнего каталога.

### ✅ Выполненные действия

1. Определил текущего пользователя системы.
2. Определил текущий каталог.
3. Перешёл в домашний каталог пользователя.
4. Проверил текущее расположение.
5. Просмотрел содержимое домашнего каталога.
6. Просмотрел скрытые файлы и каталоги.
7. Получил подробную информацию о содержимом каталога.
8. Просмотрел полный список объектов вместе со скрытыми файлами.
9. Сравнил результаты команд просмотра содержимого каталога.

### 📂 Полученная структура

Домашний каталог пользователя:

```text
/home/g04er
```

Содержимое каталога:

```text
linux_practice
portfolio
```

### 📸 Рисунок 1

**Определение пользователя и просмотр содержимого домашнего каталога**

> Вставить скриншот выполнения команд `whoami`, `pwd`, `ls -la`.

### 💻 Изученные команды

| Команда  | Назначение                        |
| -------- | --------------------------------- |
| `whoami` | Отображение текущего пользователя |
| `pwd`    | Отображение текущего каталога     |
| `cd`     | Переход между директориями        |
| `ls`     | Просмотр содержимого директории   |
| `ls -a`  | Отображение скрытых файлов        |
| `ls -l`  | Подробная информация о файлах     |
| `ls -la` | Полный подробный список файлов    |

## 🔹 Практическая работа №2

### 🎯 Цель

Научиться создавать каталоги и файлы через терминал Linux.

### ✅ Выполненные действия

1. Перешёл в домашний каталог.
2. Создал директорию `linux_practice`.
3. Перешёл в созданную директорию.
4. Создал файл `file1.txt`.
5. Создал файл `file2.txt`.
6. Проверил содержимое директории.
7. Получил подробную информацию о созданных файлах.
8. Просмотрел содержимое файла.
9. Открыл файл через программу `less`.
10. Вернулся в домашний каталог.
11. Проверил наличие директории `linux_practice`.
12. Просмотрел содержимое директории без перехода в неё.

### 📂 Полученная структура

```text
linux_practice/
├── file1.txt
└── file2.txt
```

### 📸 Рисунок 2

**Создание директории и файлов**

> Вставить скриншот выполнения команд `ls`, `ls -l`, `less file1.txt`.

### 💻 Изученные команды

| Команда | Назначение                      |
| ------- | ------------------------------- |
| `mkdir` | Создание директорий             |
| `touch` | Создание файлов                 |
| `ls`    | Просмотр содержимого директории |
| `ls -l` | Подробная информация о файлах   |
| `cat`   | Просмотр содержимого файла      |
| `less`  | Постраничный просмотр файла     |
