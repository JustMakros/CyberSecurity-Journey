# Linux Day 01

## Практическая работа №1

### Цель

Впомнить базовые команды Linux для создания каталогов, файлов и навигации по файловой системе.

### Выполненные действия

1. Создал директорию `portfolio`.
2. Перешёл в директорию `portfolio`.
3. Создал внутри неё директорию `linux`.
4. Перешёл в директорию `linux`.
5. Создал директорию `day01` для хранения материалов первого дня.
6. Перешёл в директорию `day01`.
7. Создал текстовый файл `notes.txt` для заметок.
8. Создал текстовый файл `commands.txt` для записи изученных команд.
9. Просмотрел содержимое текущей директории.
10. Определил текущий путь в файловой системе.

### Полученная структура

portfolio/
└── linux/
    └── day01/
        ├── notes.txt
        └── commands.txt

### Изученные команды

- mkdir — создание директорий
- cd — переход между директориями
- touch — создание файлов
- ls — просмотр содержимого директории
- pwd — отображение текущего пути

### Результат

Вспомнил как создавать каталоги и файлы через терминал Linux, перемещаться по файловой системе и проверять содержимое директорий.
