# Linux Day 07

📅 **Дата:** 27.06.2026

## 🛠 Практическая работа №3

### 🎯 Цель

Научиться анализировать текстовые файлы с помощью стандартных утилит Linux и создавать архивы для хранения, резервного копирования и передачи данных.

### ✅ Выполненные действия

1. Перешёл в домашний каталог пользователя.
2. Создал рабочую директорию `text_archive_practice`.
3. Создал текстовый файл `students.txt` с помощью редактора Nano.
4. Просмотрел содержимое файла командой `cat`.
5. Вывел строки файла в обратном порядке командой `tac`.
6. Просмотрел первые строки файла с помощью `head`.
7. Просмотрел последние строки файла с помощью `tail`.
8. Выполнил поиск текста командой `grep`.
9. Выполнил поиск без учёта регистра.
10. Исключил строки из результата поиска.
11. Отсортировал строки файла.
12. Подсчитал количество строк, слов и символов с помощью `wc`.
13. Просмотрел файл через `less`.
14. Просмотрел файл через `more`.
15. Создал дополнительные файлы `file1` и `file2`.
16. Создал архив `archive.tar`.
17. Просмотрел содержимое архива без распаковки.
18. Создал директорию для распаковки архива.
19. Распаковал архив в отдельный каталог.
20. Создал архив со сжатием `archive.tar.gz`.
21. Проверил наличие созданных архивов.
22. Создал директорию для распаковки сжатого архива.
23. Распаковал архив `archive.tar.gz`.
24. Проверил содержимое распакованных файлов.
25. Вернулся в домашний каталог.
26. Удалил рабочую директорию.

### 💻 Изученные команды

| Команда    | Назначение                            |
| ---------- | ------------------------------------- |
| `cat`      | Просмотр содержимого файла            |
| `tac`      | Вывод строк файла в обратном порядке  |
| `head`     | Просмотр начала файла                 |
| `tail`     | Просмотр конца файла                  |
| `grep`     | Поиск текста по шаблону               |
| `sort`     | Сортировка строк                      |
| `wc`       | Подсчёт строк, слов и символов        |
| `less`     | Постраничный просмотр файла           |
| `more`     | Постраничный просмотр файла           |
| `touch`    | Создание файлов                       |
| `tar -cf`  | Создание архива                       |
| `tar -tf`  | Просмотр содержимого архива           |
| `tar -xf`  | Распаковка архива                     |
| `tar -czf` | Создание архива со сжатием Gzip       |
| `ls`       | Просмотр содержимого каталога         |
| `mkdir`    | Создание директории                   |
| `rm -r`    | Удаление каталога вместе с содержимым |
